Setup two-factor authentication

Чтобы настроить двухфакторную аутентификацию выполните следующие шаги:

  1. В приложении выберите добавить учетную запись и выберите Time-based One-time Password TOTP
  2. Отсканируйте секретный ключ в приложении
    {{ otp_secret_key }}
    Alternatively вы можете ввести секретный ключ вручную
  3. Введите код аутентификации в форму ниже
{% csrf_token %}
